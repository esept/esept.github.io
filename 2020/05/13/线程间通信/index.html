java_线程间通信

发表于 2020-05-13 | 分类于 java

概念

多个线程在使用统一资源,但是处理的动作(线程的任务)不同

在处理统一资源的时候就存在线程通信问题

多个线程并发执行的时候,在默认条件下CPU是随机切换线程的,当我们需要多个线程来处理统一任务,并且希望他们有规律的执行,那么多线程之间需要一些协调通信,以此来帮助我们达到多线程共同操作同一份数据

多个线程在操作同一份数据时,为了避免对同一共享变量的争夺,我们需要通过<等待唤醒机制>来使各个线程能有效的使用资源

等待唤醒机制: 线程之间的通信

重点: 有效的利用资源(吃一个,做一个,吃一个,做一个)
